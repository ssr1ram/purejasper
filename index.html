What is Purejasper?

Unlike Word and Google docs where you author in a linear paragaraph, section, page type manner. Purejaspers let you mix this up, so you can author paragraphs that link to each other making navigation between them easy.

How is this different from a wiki?

Similar to wikis, Purejaspers help you build a knowledge base on a given topic over time. Unlike wikis which need third party hosts and server engines, Purejaspers use your Dropbox files and Browser making it private and self-contained.

What do folks use Purejasper's for?

Planning an event and/or undertaking a project: A Purejasper helps you collect and organize all the disparate bits of information you need. From links to the various web services you would be using to ideas, thoughts and other important information.
